Go by Example

この文書は、Go by Example の日本語訳となります。

Go は、オープンソースの プログラミング言語です。スケールしやすく、安全な、信頼性の高い ソフトウェアを開発できるように設計されています。詳しく知るには、 公式ドキュメント を読んでください。

Go by Example は、コメント付きの サンプルプログラムを使った、ハンズオン形式の Go 入門です。 最初のサンプル に進むか、 下のリストから読んでください。

特にことわりがない限り、ここでのサンプルは Go の最新メジャーリリース を前提としており、新しい言語機能を使っている可能性があります。うまく動作しない場合は、 最新版にアップグレードしてください。
