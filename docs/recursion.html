Go by Example: Recursion

Go は、 再帰関数 (recursive functions). をサポートします。これは典型的な例です。

 
package main

import "fmt"

この fact 関数は、終了条件となる fact(0) に達するまで、自分自身を呼び出します。

 
func fact(n int) int {
    if n == 0 {
        return 1
    }
    return n * fact(n-1)
}

func main() {
    fmt.Println(fact(7))

        }
        return fib(n-1) + fib(n-2)
